# complete the Word Game

A browser based word guessing game built using vanilla JavaScript, HTML, and CSS.  
Features include dynamic word selection, score tracking and interactive UI.
ui colour pallet: #f1eaff #e7bcde #bb9cc0 #67729d
includes an assistant that talks to you while in game!

## how to run
Open `index.html` in any modern browser.

## what i learned
- DOM manipulation and event handling  
- Basic game logic and state management  
- Responsive UI design
